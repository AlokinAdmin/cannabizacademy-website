# Alokin Brand Guidelines

## Logo assets

| File | Usage |
|---|---|
| `logomark.png` | Primary mark — brushed triangle only. Use on white or very light surfaces. Minimum 32px. |
| `logo-tagline.png` | Horizontal lockup — mark + ALOKIN wordmark. Widescreen horizontal applications. |
| `logo-intelligence-trust.jpeg` | Full lockup — mark + ALOKIN + "INTELLIGENCE & TRUST LAYER" descriptor. Use when introducing the brand for the first time in a context; otherwise prefer the simpler lockups. |

## Mark construction

Brushed, open triangle. Three arcs:
- **Top-left → top:** Lime green `#8FC63F`
- **Top-right → bottom-right:** Deep violet `#4A3F8F`
- **Bottom-right → bottom-left:** Teal/cyan `#3BA9C8`

Organic, painterly edges — never redraw with geometric / hard-edge construction. Always use the PNG asset.

## Clear space

Minimum clear space around the mark equals the height of the inner negative triangle. Do not crop.

## Placement on dark surfaces

The mark has transparent background and reads well on black. The tagline lockup uses black wordmark type — **do not use `logo-tagline.png` on dark backgrounds**; use `logomark.png` + custom light-type wordmark instead.

## Voice positioning

"Intelligence & Trust Layer" — the descriptor should appear with the full-lockup file (`logo-intelligence-trust.jpeg`) or be set in the brand's type styles when paired with the simpler mark.
